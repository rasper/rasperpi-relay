RasperPi-Relay
============

## Quickstart

1. Clone the repo

        git clone https://github.com/rasper/rasperpi-relay.git
        cd rasperpi-relay

2. Create a new `settings.py` file based on `settings.py.sample` and modify if necessary

        cp settings.py.sample settings.py

3. Start the relay

        sudo python3 rasperpi-relay.py
